# web-automation
A web automation to get phonetic from website

all can be done by:

1. open an excel table and count how many lines we've got here
2. start a web automation, will use Chrome drive
3. visit https://tophonetics.com/zh/
4. pick a word from the table and fill it in the website, then click the button
5. get the result and fill it back in the table
6. automatically start a new inquiry
